# Test Task: Hashnode Engagement-Reviewer Sync Verification

**Purpose:** Verify that hashnode engagement and engagement-reviewer are syncing correctly.

**Task:** Create an engagement plan for interacting with trending Hashnode posts.

**Scenario:**
- Found 5 trending posts in #webdev tag
- All have 10+ reactions
- Need to decide: engage or skip?

**Requirements:**
- Analyze each post's topic and quality
- Select 2-3 posts worth engaging with
- For each: should we react, comment, or follow author?
- Draft genuine comment (if applicable)
- Explain selection criteria

**Expected Outputs:**
1. Engagement output file with strategy
2. Engagement-reviewer verdict file (APPROVED/REVISION_NEEDED)
3. state.json updated with task completion
4. MEMORY.md updated with any learnings

**Success Criteria:**
- Both output files created
- Engagement decisions are well-reasoned
- Comments (if any) add value
- Reviewer validates authenticity
- Full cycle completes in < 5 minutes
